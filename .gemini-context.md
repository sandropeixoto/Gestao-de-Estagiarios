# context_seed

Este projeto é o 'Gestão de Estagiários', uma plataforma separada em Frontend (React.js, Vite, Tailwind CSS, Lucide React, Axios) e Backend (Node.js, Express.js, TypeScript), utilizando Supabase (PostgreSQL) como banco de dados. A arquitetura segue o padrão MVC no backend e Componentes Inteligentes/Burros no frontend, respeitando princípios SOLID e Clean Code, com nomenclatura `PascalCase` para componentes React e `camelCase` para funções/arquivos backend. O Design System é baseado em 'Plus Jakarta Sans', com paleta focada em roxo (`#7C3AED` e `#A78BFA`), fundo lavanda claro (`#FAF5FF`), uso de sombras suaves, cantos arredondados generosos (`rounded-xl`) e micro-interações claras. As regras de negócio centram-se na união obrigatória de Estudante (CPF único), Instituição (CNPJ único e status de convênio) e Supervisor para a criação de um Contrato, que por sua vez gerencia módulos periféricos como Folha de Ponto (com geolocalização e flag de dias de prova), Avaliações de Desempenho e Acertos Financeiros (saldo de recesso).
