# 📖 Guia do Usuário - EstagiárioPlus

Este manual descreve como utilizar as principais funcionalidades do sistema de Gestão de Estagiários.

---

## 1. Primeiro Acesso
*   Acesse o sistema via link institucional.
*   O login é realizado através de **SSO (Single Sign-On)**.
*   Após o login, você verá o Dashboard com os principais KPIs.

## 2. Fluxo de Trabalho (Workflow)

### Etapa 1: Cadastros Base
Antes de criar um contrato, certifique-se de que os seguintes cadastros estejam atualizados:
1.  **Estudantes:** No menu `Estudantes`, clique em `Novo` e preencha os dados (CPF, Curso, Banco).
2.  **Instituições:** Verifique se o órgão concedente está cadastrado e com convênio `Ativo`.
3.  **Supervisores:** Garanta que o supervisor da unidade destino esteja cadastrado na subunidade correta.

### Etapa 2: Abertura de Vaga
*   Vá em `Vagas` e crie uma nova posição.
*   Defina o valor da bolsa, auxílio transporte e a lotação exata (Unidade/Subunidade).
*   Uma vaga só pode ser aberta se houver pelo menos um supervisor na subunidade.

### Etapa 3: Formalização do Contrato
*   O contrato é o vínculo central. No menu `Contratos`, clique em `Novo`.
*   Selecione o Estudante e a Vaga.
*   O sistema preencherá automaticamente os dados financeiros da vaga.
*   Informe a data de início, fim e os dados do seguro.
*   Após salvar, o contrato estará `Ativo`.

---

## 3. Gestão Diária

### 🕒 Folha de Ponto
*   Os estagiários registram entrada e saída.
*   **Geolocalização:** O sistema captura onde o ponto foi registrado para fins de compliance.
*   **Dia de Prova:** Em dias de prova, marque a opção correspondente para que a jornada seja reduzida pela metade conforme a Lei do Estágio.

### 📈 Avaliações
*   Periodicamente, acesse o módulo de `Avaliações`.
*   O supervisor deve preencher o feedback técnico e comportamental.
*   A nota gerada ajuda no acompanhamento do desenvolvimento do estagiário.

---

## ❓ Perguntas Frequentes (FAQ)

**P: Não consigo vincular um estudante a uma vaga.**
*R: Verifique se a vaga já atingiu a capacidade máxima (`quantidade`) ou se o contrato do estudante está ativo em outra vaga.*

**P: O supervisor não aparece na lista de contratos.**
*R: O supervisor deve estar vinculado à mesma Unidade/Subunidade da vaga selecionada.*
